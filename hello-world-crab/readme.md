# Installation
```bash
cd /tmp
wget https://github.com/yanc0/crabe-life/releases/download/v1.0/crabe-life_v1.0_linux_amd64
chmod +x crabe-life_v1.0_linux_amd64
sudo mv crabe-life_v1.0_linux_amd64  /usr/local/bin/crabe-life
crabe-life -crab-url https://i.giphy.com/media/aoegxxFogBQm74XCWY/giphy.webp
```
